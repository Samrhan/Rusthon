# Getting Started

This section will help you get Rusthon up and running on your system.

## Prerequisites

Before you begin, ensure you have:

- **Rust** (stable toolchain, 1.70+)
- **LLVM 18** with development libraries
- **Clang 18** compiler
- **Git** for cloning the repository

## Platform Support

Rusthon has been tested on:
- ✅ **Ubuntu 22.04+** (recommended)
- ✅ **macOS** (with Homebrew)
- ✅ **Arch Linux**
- ⚠️ **Windows** (via WSL2 recommended)

## Development Container (Recommended)

The easiest way to get started is using the provided devcontainer:

```bash
# Clone the repository
git clone https://github.com/Samrhan/Rusthon.git
cd Rusthon

# Open in VS Code with Remote-Containers extension
code .
# Then: "Reopen in Container" from the command palette
```

The devcontainer includes all necessary dependencies pre-configured.
